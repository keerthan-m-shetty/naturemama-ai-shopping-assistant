# NatureMama Heritage вҖ“ Project Structure

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS with custom brand tokens
- **Backend**: AWS Amplify Gen 2 (Auth + Data/AppSync + S3 Storage)
- **Fonts**: Inter (sans) + Playfair Display (serif) via next/font
- **Carousel**: Embla Carousel
- **Icons**: Lucide React

## Data Schema Summary (`amplify/data/resource.ts`)

| Model                  | Purpose                                      | Auth                          |
|------------------------|----------------------------------------------|-------------------------------|
| `Product`              | Product catalog with line, badges, media     | Public read / Admin write     |
| `ProductVariant`       | Size/format variants with SKU & price        | Public read / Admin write     |
| `Review`               | Customer reviews with moderation flag        | Owner write / Public read     |
| `Cart`                 | Per-user cart container                      | Owner only                    |
| `CartItem`             | Line items with price snapshot               | Owner only                    |
| `Order`                | Immutable order record post-checkout         | Owner read / Admin full       |
| `OrderItem`            | Order line items with purchase-time snapshot | Owner read / Admin full       |
| `NewsletterSubscriber` | Email capture from homepage/quiz             | Guest create / Admin full     |
| `QuizResult`           | Quiz answers + recommended product line      | Guest create / Owner read     |

## Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Deploy Amplify backend (requires AWS credentials)
npx ampx sandbox

# 3. Download amplify_outputs.json from the console and place in project root

# 4. Start dev server
npm run dev
```

Visit http://localhost:3000
